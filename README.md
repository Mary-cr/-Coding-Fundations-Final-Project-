 Coding Foundations Final Project
A HTML & CSS replica of the CodePen.io homepage, created as the final project for CS100 Coding Foundations course.

 Features
- Navigation bar with "Pens", "Projects", "Posts", "Collections", and "Spark" links
- Responsive layout matching the original CodePen design
- Images and visual elements recreated using CSS
- Accurate color scheme and typography

How to View:
Simply open `index.html` in your web browser to view the project.
